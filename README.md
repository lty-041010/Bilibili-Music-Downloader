# Bilibili-Music-Downloader
批量下载 B 站收藏夹视频并自动转换为 MP3 音频，用于个人离线听歌。

---

## 快速开始
1. 下载本仓库代码
2. 从 [Release 附件](https://github.com/lty-041010/Bilibili-Music-Downloader/releases) 下载 `ffmpeg.exe`
3. 将 `ffmpeg.exe` 放到和 `源码` 同目录
4. 运行软件即可

---

## 配置要求
- 操作系统：Windows 10 / Windows 11（64 位）
- Python 版本：Python 3.11.x（推荐 3.11.4 ~ 3.11.9）
- 必备工具：ffmpeg（用于音频格式转换）
