B 站音频下载器 完整使用 & 配置说明

直接使用：完成四、五、六步骤后运行main.py

一、配置要求（运行 / 打包必备）
操作系统：Windows 10 / Windows 11（64 位）
Python 版本：Python 3.11.x（推荐：3.11.4 ~ 3.11.9，稳定无兼容问题）
必备工具：ffmpeg（音频转换）
硬件：普通办公电脑即可，无特殊配置要求

二、Python 运行库安装（源码运行 / 打包必装）
# 1. 安装项目核心运行依赖（固定版本，稳定不报错）
pip install PyQt6==6.6.0 requests==2.31.0

# 2. 安装打包工具（仅需要打包exe时安装）
pip install pyinstaller==6.4.0


三、打包为 Windows 桌面软件（一键命令）

pyinstaller -w -i app.ico -n B站音频下载器 main.py

✅ 打包后软件位置：项目文件夹 dist/B站音频下载器/
✅ 软件图标、名称、无黑框控制台，全部配置完成

四、准备工作（必须）
下载 ffmpeg.exe，放到和软件同目录（和 exe / 主程序放一起，否则无法转成 MP3）
下载地址：https://ffmpeg.org/download.html （Windows 版本）

五、获取登录凭证（SESSDATA /bili_jct）
打开 B 站网页端，登录你的账号
按 F12 打开开发者工具 → 切换到「应用程序」标签
在左侧找到「Cookie」→ https://www.bilibili.com
找到 SESSDATA 和 bili_jct 两个字段，复制它们的值（注意不要复制前后的空格）
粘贴到软件的对应输入框里即可，软件会自动保存
⚠️ 注意：SESSDATA 有效期约 1 个月，过期后重新按上面步骤更新即可

六、获取收藏夹 ID（fid）
进入你的 B 站个人主页 → 打开「收藏夹」
看浏览器地址栏，链接类似：https://space.bilibili.com/xxx/favlist?fid=123456789
其中 123456789 就是你的收藏夹 ID，复制粘贴到软件对应输入框

七、使用步骤
点击「加载收藏夹」，软件会读取你的收藏夹视频列表
用 Ctrl/Shift 选择你要下载的视频
点击「选择保存目录」，设置你想保存的位置（软件会自动记住）
点击「批量下载选中音频」，等待完成即可

八、常见问题
下载失败 / 403 错误：检查 Cookie 是否过期、账号是否有视频权限（大会员 / 充电视频无法下载）
无法转成 MP3：确认 ffmpeg.exe 和软件在同一个文件夹里
找不到下载的文件：打开软件，点击「选择保存目录」，弹出的文件夹就是当前保存位置
打包报错：必须使用 Python 3.11 版本，其他版本可能不兼容

九、开源说明
本项目仅用于学习交流，请勿用于商业用途
仅支持下载公开、无版权限制的 B 站视频音频
付费 / 充电专属视频受版权保护，无法下载
